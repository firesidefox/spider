设计要点

  • 布局: Chat + 目标视图（方案 B）
  • 对话风格: Claude Code 终端风格（等宽字体、❯ prompt、深色背景）
  • 目标视图: 混合方案（方案 C）— 列表/矩阵切换 + 统计摘要 + 失败置顶
  • Tool 调用: 可折叠条，不干扰主对话流
  • 确认机制: 内联确认栏，标注 risk level
  • 对话切换: 顶部栏下拉，不占常驻空间
